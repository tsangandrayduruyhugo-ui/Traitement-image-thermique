# Traitement-image-thermique
Traitement d’images thermiques et analyse des profils thermiques avec Python (OpenCV, NumPy) pour des applications en génie mécanique.
# Analyse d’images thermiques avec Python

##  Description
Ce projet consiste à traiter des images thermiques et à extraire des profils de température en utilisant Python.

##  Objectifs
- Traiter des images infrarouges
- Extraire des profils thermiques
- Effectuer une analyse statistique
- Détecter automatiquement les points chauds

##  Outils utilisés
- Python
- OpenCV
- NumPy
- Matplotlib

## Fonctionnalités
- Extraction de profil thermique
- Calcul du gradient thermique
- Analyse statistique
- Détection des anomalies thermiques

##  Application en génie mécanique
- Surveillance des machines
- Analyse des pertes de chaleur
- Détection de défauts

##  Améliorations futures
- Analyse en temps réel
- Interface graphique
- Intégration avec caméra thermique
